# run-chatbot

Run the interactive chatbot to test the LangGraph workflow.

## Usage
```bash
uv run main.py
```

## What it does
- Starts the interactive CLI chatbot
- Accepts user messages and routes them through the graph
- Displays therapist or logic agent responses based on message classification
- Type "exit" to quit

## Common use cases
- Testing message classification
- Verifying agent routing works correctly
- Manual end-to-end testing
